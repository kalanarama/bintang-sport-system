# 🏸 Bintang Sport Center
Sistem Informasi Booking Lapangan Olahraga Berbasis Web

## Fitur
- 🔐 Login Admin
- 🏟️ Kelola Data Lapangan
- 📅 Kelola Jadwal
- 🎉 Kelola Promo & Diskon
- 📦 Booking Online tanpa akun
- 💳 Pembayaran via QRIS
- 📊 Laporan Rekapitulasi & Export PDF
- 🔔 Notifikasi WhatsApp
- 🔍 Cek Status Booking

## Tech Stack
- **Frontend**: Bootstrap, HTML, CSS, JavaScript
- **Backend**: Laravel (PHP)
- **Database**: MySQL
- **Desain UI**: Figma

## Instalasi
1. `git clone https://github.com/kalanarama/bintang-sport-system.git`
2. `composer install`
3. `cp .env.example .env`
4. `php artisan key:generate`
5. Isi konfigurasi database di `.env`
6. `php artisan migrate`
7. `php artisan serve`
